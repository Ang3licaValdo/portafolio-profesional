KGProject

Knowledge Graph creado utilizando información extraída de los XML de 30 distintos artículos científicos que yo y mi equipo recolectamos, este KG se fue construyendo haciendo uso de librerías de Python, además se hicieron consultas al sitio web de ROR para enriquecer el grafo con información de las instituciones mencionadas y consultas a Wikidata para enriquecerlo con información de los autores.En este proyecto me encargué de desarrollar funciones para extraer información del XML y para las requests a los sitios web de lo que se obtuvo información, además, también me encargué de la documentación y de pasar todo el código a un contenedor de Docker.

Software for data visualization obtained with text analysis tools:

Es un software que consiste de tres distintos scripts de Python que tienen como objetivo que 10 artículos científicos en formato pdf se conviertan en XMLs para así poder obtener información específica de cada uno de estos, cada script tiene una función específica, el primero retorna una nube de palabras con las palabras claves de los abstract de los 10 artículos, el segundo retorna una gráfica mostrando el número de figuras presentes en cada uno de los artículos y, por último, el tercer script crea una lista de todos los links encontrados en los artículos y la devuelve en un .txt. Este software se ejecuta en un contenedor de docker y usando docker compose para que los archivos se ejecuten al mismo tiempo, haciendo uso de un volumen compartido para que el usuario pueda ver los archivos devueltos en su computadora.
